Recommendation System

Jupyter Notebook & Python

GitHub Repo Live version

The Jupyter Notebook Recommendation System is a data science project developed with Python. It leverages a dataset of 130 U.S. hospitals from 1999-2008 for patients with diabetes to create a predictive model. The project includes an exploratory data analysis, feature engineering, and implementation of machine learning algorithms to predict patient readmission. The technologies used include Numpy, Pandas, Seaborn, Matplotlib, and more.
