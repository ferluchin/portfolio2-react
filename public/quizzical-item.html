Quizzical Game

React.js & OTDB API

GitHub Repo Live version

The Trivia Quiz is an interactive game that pulls questions from the OTDB API. It contains two screens: start & questions. The objective of the game is to answer the trivia questions correctly. The score is tallied after the "Check answers" button is clicked. Built using React.js.
